## **5.** 🥇**회고**

---

- React 및 CSS에 대한 이해도가 낮아 개발 과정에서 다소 어려움을 겪었으며, 프론트엔드 전반에 대한 기술 역량 향상의 필요성을 절감함
- React를 처음 접한 만큼, 컴포넌트 구조와 상태 관리 등 기초 개념부터 체계적으로 학습할 계획임
- 프로젝트를 통해 JavaScript 기반 프레임워크를 다루기 위해서는 JS 자체에 대한 탄탄한 이해가 선행되어야 함을 실감함
- SCSS를 활용한 디자인 작업이 예상보다 복잡하고 세밀한 조정이 요구되어, 향후 다양한 실습을 통해 UI/UX 구현 능력을 강화할 필요가 있음

## 🥇**5-1.** 🎆**한계점**

- 제한된 시간으로 인해 세부적인 완성도가 다소 미흡하였으며, 일부 UI 요소의 정교한 구현이 부족함
- 메인 화면을 포함한 전체 페이지의 주요 기능 구현이 일부 완료되지 못함
